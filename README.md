# Solana Security Reports

I'm a Rust developer learning Solana security. Here I keep my own audit reports, CTF writeups, and notes on vulnerabilities I've found along the way or learnt from experienced security reserchers.

## Structure

- `audit_arena/` - reports from [Solana Audit Arena](https://github.com/Frankcastleauditor/Solana-Audit-Arena)
- `solctf/` - writeups from [sol-ctf](https://github.com/sol-ctf/public-bank)
